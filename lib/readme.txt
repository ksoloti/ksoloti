3rd party library usage


JGIT - git api - used for syncing user library with GIT
SL4J - Logging library, used by JGIT
JSCH - ssh security, used by JGIT
libusb4java - Native JNI wrapper for libusb
flatlaf - modern UI
flatlaf-intellij-themes - for modern UI
rsyntaxtextarea, autocomplete - object editor code syntax
jsvg - handle SVG icons
batik - SVG graphic tools
jfilechooserbookmarks - simple little bookmarks manager inside the file chooser
