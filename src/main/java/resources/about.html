Get involved!

Axoloti was conceived as a community project, where members could help and inspire each other to make the most of Axoloti.

Please join the community forum at http://community.axoloti.com, where you will also find announcements, latest update and support for Axoloti.
Edit: The above link is probably inaccessible. There is a read-only backup of the forum content at http://sebiik.github.io/community.axoloti.com.backup.

If you're on Discord, feel free to join the Ksoloti Discord Server.

For DIY resources check out the official Ksoloti homepage at ksoloti.github.io.

  • ChibiOS
  • GCC-arm-none-eabi
  • LibUSB
  • Usb4java
  • dfu-util
  • Simple-xml
  • Java
  • Marlin Renderer for Java 17+
  • FlatLaf - Swing Look and Feel
  • RSyntaxTextArea
